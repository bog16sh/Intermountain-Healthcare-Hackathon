AI-Powered Event Intake & Workflow Automation System
Built for the Intermountain Healthcare Generative AI Hackathon 2026

Developed within a 6-hour hackathon under conditions with no prior access to the problem context or requirements, this project is an end-to-end conversational AI system that automates the intake, validation, and routing of community health event and material requests.
It replaces a manual, email-based workflow with a structured, intelligent pipeline that improves data quality, reduces administrative burden, and enables scalable operations.

The repository includes the presentation delivered to Intermountain Healthcare judges.
